# Chat GPT Extension

The Chat GPT extension loads the **Chat GPT web and integrates it with Visual Studio 2022.**

It works with both FREE and Plus versions of Chat GPT web. **No OpenAI tokens are required.**

Also integrates with **Google Gemini** FREE and Advanced.

## Features

- Send selected code from VS.NET to GPT/Gemini
- Send selected code from GPT/Gemini to VS.NET
- When the GPT/Gemini "Copy code" button is clicked, the code is sent automatically to VS.NET
- Buttons to automate code fixing, improve code and asking GPT/Gemini to show complete code
- Other code operations like commenting code, adding exception handling, creating unit tests, and several others
- Configurable custom actions for GPT/Gemini to apply to the code; these can be added, removed, changed, or reordered as needed
- Last AI model used will be saved for next sessions

## Install

1. Close VS.NET
2. Download and install the extension
3. Open VS.NET
4. **Go to the menu View -> Other Windows and select Chat GPT Extension**
5. Log into Chat GPT

## Uninstall

1. In VS.NET, go to extensions -> manage extensions, click on installed, select and remove the Chat GPT Extension
2. Close VS.NET to apply changes
3. Delete the directory %appdata%\\..\Local\ChatGPTExtension

## Usage

- Use CTRL+mouse scroll to zoom in or out in the extension for better layout of the web site

- The "VS.NET to GPT/Gemini" button copies the selected code from VS.NET to the GPT/Gemini prompt area, preserving any existing data.

- The "Fix Code in GPT/Gemini" button copies the selected code from VS.NET to the GPT/Gemini prompt area and asks AI to fix it.

- The "Improve Code in GPT/Gemini" button copies the selected code from VS.NET to the GPT/Gemini prompt area and asks AI to improve it.

- The "GPT/Gemini to VS.NET" button copies the selected text from GPT/Gemini and sends it to VS.NET.

- The "Complete Code" button asks GPT/Gemini to show full complete code when it is showing partial code only.

- The "Enable Copy Code" switch, when enabled, automatically sends the code to VS.NET when the "Copy code" button in GPT/Gemini is clicked.

- The "Arrow" button opens a submenu with other code operations that GPT/Gemini can perform.

- The "Reload Chat GPT/Gemini..." submenu reloads GPT/Gemini in the internal browser.

- The "Configure extension..." submenu opens the screen to configure the code action operations.

- The "Use GPT" submenu will configure the extension to use Open AI Chat GPT

- The "Use Gemini" submenu will configure the extension to use Google Gemini

## Known Issues

There are some known random issues with logging off from Chat GPT. If you encounter problems logging off, try pressing F12 and then trying again, or close VS.NET and delete the directory %appdata%\\..\Local\ChatGPTExtension.
